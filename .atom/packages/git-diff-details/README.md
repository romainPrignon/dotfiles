# git-diff-details package

View git diffs directly in atom.

Keybindings:
  * `alt-g alt-d` to toggle the diff view
  * `escape` to close the diff view
  * `alt-u` for undo
  * `alt-c` for copy

You can choose whether the diff should be highlighted or not:
